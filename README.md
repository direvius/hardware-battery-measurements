# hardware-battery-measurements
Измерение потребления батарейки телефоном - arduino-way.  
  
Как использовать:  
1. Скомпилить.  
2. Подключить arduino к USB порту  
3. Запустить serial-reader. 
  
help почитать так: serial-reader --help
